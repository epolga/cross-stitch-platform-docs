# WPF Uploader Integration Planning

This folder contains planning documents for integration between the WPF uploader and:
- The cross-stitch web application (upload API, metadata sync, status updates)
- AWS (S3 uploads, authentication, DynamoDB if used)
- Pinterest (board/pin creation, metadata automation)

## Key Integration Points
- Upload process and error handling
- Data contracts for uploads and metadata
- Synchronization with the web app
- S3 path conventions and authentication
- Pinterest automation flows
- Versioning and change management
