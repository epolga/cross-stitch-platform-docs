# Uploader Documentation Folder Structure

- docs/uploader/templates: Email templates (HTML and text)
- docs/uploader/helpers: Helper class documentation
- docs/uploader/images: Visual assets
- docs/uploader/bin: Build outputs (for documentation only)
- docs/uploader/obj: Intermediate build files (for documentation only)

All non-planning, non-roadmap files related to Uploader are now under docs/uploader. Planning and roadmap docs remain in plan/.
